# Labyrinth Navigation Robot – Webots

This project implements an autonomous maze-solving controller for a Thymio2 robot in Webots.

## Project Overview

The goal was to program a robot to navigate through a labyrinth without touching the walls while maintaining a stable distance from the walls on straight sections.

The robot uses:
- Three distance sensors: left, front, and right
- Compass heading feedback
- Real-time motor control
- Maze navigation logic

## Robot Platform

- Robot: Thymio2
- Simulator: Webots
- Sensors:
  - Left distance sensor
  - Front distance sensor
  - Right distance sensor
  - Compass sensor

## Control Objective

The robot must:
- Navigate through the maze
- Avoid wall contact
- Maintain approximately constant wall distance
- Choose a fast solving strategy
- Complete the maze autonomously
